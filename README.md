# Vuefire CRUD

The simplest way to create CRUD + PDF Export using Vue.js, Firebase, Vuefire, Vuetify, jsPDF & jsPDF-autoTable.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
